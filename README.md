CEA_FlujoArticulos

Este proyecto implementa el flujo CRUD de Articulos en un sistema bibliográfico siguiendo la arquitectura Hexagonal.
Incluye:

Implementación en Java

Conexión mediante JDBC a PostgreSQL

Scripts de base de datos

Instalación y ejecución del sistema

Evidencias para la Actividad 4


BASE DE DATOS.

Base utilizada: DesarrolloU4
Motor: PostgreSQL, PGADMIN 4

El script DesarrolloU4.sql se encuentra en la raíz del proyecto.
Este archivo contiene la definición de la tabla ARticulos y datos de prueba realizados en ella.


CONFIGURACION DEL ENTORNO.

Instalar PostgreSQL

Usar su motor PGADMIN 4

Crear la base de datos DesarrolloU4

Ejecutar el script SQL

Editar credenciales en:

src/infraestructure/ArticuloRepositoryPostgres.java


EJECUCION.

Directamente desde Netbeans con el Run

OPERACIONES CRUD DISPONIBLES.

Crear Artículo

listar Artículos

Buscar Artículo por ID

Actualizar Artículo

Eliminar Artículo
